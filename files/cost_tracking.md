Project Cost Tracking

The platform tracks project cost using supplier contracts and variation costs.

Cost components
Component	Description
Supplier Cost	Cost from supplier proposals
Variation Cost	Additional approved work
Margin	Revenue minus supplier cost
Margin formula

project_margin = project_revenue − total_supplier_cost

Margin recalculation triggers

Margin is recalculated when:

a supplier proposal is approved

an addendum is created

a variation cost is approved

Important note

Costs are associated with units because units represent sellable properties.

This ensures accurate margin tracking per property.
