Supplier Workflow

Construction projects involve external suppliers.

Typical suppliers include:

electricians

plumbers

facade contractors

roofing specialists

insulation installers

Work items

Projects define work items that describe tasks requiring suppliers.

Examples:

facade installation

plumbing

electrical wiring

roofing

Suppliers submit proposals for these work items.

Proposal fields

A proposal contains:

supplier identifier

work item reference

estimated cost

estimated timeline

optional technical notes

Proposal approval

Project managers review proposals.

Once approved, the proposal becomes the active supplier contract.

Rejected proposals remain stored.

Price changes

Prices may change before approval.

After approval the price is normally fixed.

However, addendums may introduce additional work or cost adjustments.
